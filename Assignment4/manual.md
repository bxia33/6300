# **User Manual**

This is an android application for distance converter.

## **Required input**
* Users shall type a number into the "value" field. 
* Users need to choose the distance unit that he/she wants to convert. The left side are the distance unit that users want to convert from, the right side are the distance units that users want to convert to.

## **Expected output**
* After click button of "Convert", the app will output result at "Result" field, if “from" unit and "to" unit are same, the value won't change. If there is a conversion between units and the result is less than 0.004, the result will be shown as zero.
* If the user want to reset all the input, he/she could press the "Reset" button, and then everything will be reset to default.
